# Pulmonary-Abnormalities-Detection-Using-Machine-Learning-and-Django
Pulmonary Abnormalities Detection is a system using Machine Learning technologies which can detect pulmonary abnormalities from the x-ray images. It can detect two diseases such as pneumonia and tuberculosis. This project is done as my last year final project.
Technologies: Python, Django, Tensorflow, Numpy, Keras, Pandas, Sklearn, HTML, CSS
